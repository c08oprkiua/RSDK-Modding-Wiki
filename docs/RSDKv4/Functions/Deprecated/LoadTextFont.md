# LoadTextFont

!!! failure "Deprecated"
    This function was removed from RSDKv4 in REV02.

## Description
Loads a bitmap font from `filePath` for use with text menus.

## Parameters
`filePath`

:   The path to the font.

## Return Value
None.

## Syntax
```
LoadTextFont(string filePath)
```

## Example
```
LoadTextFont("Data/Game/SystemText.gif")
```
